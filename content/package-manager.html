Software development package managers usage guide

Operating system package managers

  1. Linux
    • apt (Debian, Ubuntu) -> apt install package-name
    • yum (CentOS, Fedora) -> yum install package-name
  2. macOS
    • homebrew -> brew install package-name

Programming language package managers

  1. JavaScript
    • npm -> npm install package-name
    • yarn -> yarn add package-name
  2. Python
    • pip -> pip install package-name
  3. Java
    • Maven -> mvn install (through pom.xml)
